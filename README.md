# Zumaan

A text-based implementation of the classic tile-matching puzzle video game **Zuma**, written in x86 Assembly Language (MASM).

## üéÆ Overview

Zumaan recreates the core mechanics of the popular arcade game within a console environment. The player controls a rotating shooter (the "bandook") to fire colored balls at a chain of incoming balls moving along a set path. The objective is to eliminate all balls by creating matches of three or more of the same color before they reach the end of the path.

## ‚ú® Features

- **Classic Gameplay**: Shoot and match colored balls to clear the chain.
- **Dynamic Path**: Balls follow a predefined spiral path towards the center.
- **Score System**: Real-time scoring as you eliminate groups of balls.
- **Console Graphics**: rich ASCII art and text characters used to render the game interface, menus, and animations.
- **Menu System**: Interactive main menu with options for starting the game, viewing instructions, and developer credits.
